# 🧮 BMI Calculator ⚖️📏

A simple and accurate BMI (Body Mass Index) calculator that computes BMI based on height and weight.

---

## ✨ Features
- 📏 Calculate BMI using height and weight
- ⚖️ Fast and accurate calculation
- 🧠 Beginner-friendly and clean code
- 🚀 Lightweight and easy to use

---

## 🧪 Formula
BMI is calculated using the following formula:

BMI = weight (kg) / (height (m) × height (m))

---

## 📥 Input
- Height (meters)
- Weight (kilograms)

## 📤 Output
- Body Mass Index (BMI) value

---

## 🛠️ Usage
1. Enter your height
2. Enter your weight
3. Get your BMI result instantly 🎯

This project is built for learning, practice, and demonstrating basic programming logic.
